Conexión con mi-proyecto-backend

Integración de tu frontend en Vercel con tu repositorio de backend

Repositorio de backend: https://github.com/IgnaFull/mi-proyecto-backend.git Público
Frontend en Vercel: https://mi-proyecto-beta.vercel.app/ Necesita conexión al backend

Pasos para conectar tu backend

1

Despliega tu backend

Tu repositorio de backend necesita estar desplegado en un servicio como Vercel, Heroku, Railway o Render.

https://github.com/IgnaFull/mi-proyecto-backend.git
2

Configura las variables de entorno

En tu backend, configura la conexión a MongoDB Atlas con las variables adecuadas.

// Ejemplo de variables de entorno en el backend
MONGODB_URI=mongodb+srv://usuario:password@cluster-proyecto-els.mongodb.net/mi-proyecto
PORT=5000
FRONTEND_URL=https://mi-proyecto-beta.vercel.app
3

Actualiza tu frontend

Modifica tu frontend para que apunte a la URL de tu backend desplegado.

// En tu frontend, cambia la URL del API
const API_URL = 'https://mi-proyecto-backend.vercel.app/api';

// Ejemplo de función para obtener datos
async function getData() {
  const response = await fetch(`${API_URL}/comments`);
  return await response.json();
}
4

Configura CORS en el backend

Asegúrate de que tu backend acepte peticiones desde tu dominio de frontend.

// En tu server.js o app.js del backend
const cors = require('cors');

// Configura CORS para aceptar requests desde tu frontend
app.use(cors({
  origin: 'https://mi-proyecto-beta.vercel.app',
  methods: ['GET', 'POST', 'PUT', 'DELETE']
}));

Ejemplo de Conexión Completa

Código para conectar tu frontend con tu backend y base de datos:

// URL de tu backend (reemplaza con tu URL real)
const BACKEND_URL = 'https://mi-proyecto-backend.vercel.app';

// Función para obtener comentarios desde tu base de datos
async function getComments() {
  try {
    const response = await fetch(`${BACKEND_URL}/api/comments`);
    if (!response.ok) {
      throw new Error('Error en la respuesta del servidor');
    }
    const comments = await response.json();
    return comments;
  } catch (error) {
    console.error('Error al obtener comentarios:', error);
    throw error;
  }
}

// Uso de la función
getComments()
  .then(comments => {
    console.log('Comentarios obtenidos:', comments);
  })
  .catch(error => {
    console.error('Error:', error);
  });
