Instrucciones para ejecutar el backend

  1. Abre una terminal y navega hasta la carpeta de tu backend: cd mi-proyecto-backend
  2. Asegúrate de tener Node.js instalado (versión 14 o superior)
  3. Instala las dependencias: npm install
  4. Inicia el servidor: npm start o node server.js
  5. El backend debería estar ejecutándose en: http://localhost:3001
  6. Si el puerto 3001 está en uso, puedes cambiarlo en el archivo de configuración del backend
